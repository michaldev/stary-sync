Stary-sync

Repozytorium zawiera odtworzony styl starego wyglądu strony aliorsync.

Download this project as a .zip file Download this project as a tar.gz file

Witaj, tutaj zmienisz wygląd t-mobile usługi bankowe - na dawny wygląd alior sync.

Aby przywrócić dawny wygląd strony alior sync musisz wykonać poniższe kroki.

Chrome:

  • Pobieramy i instalujemy wtyczkę -https://chrome.google.com/webstore/detail/stylebot/oiaejidbmkiecgbjeifoejpgmdaleoha

  • Ściągamy paczkę - https://github.com/michal0468/stary-sync/zipball/master, rozpakowujemy i kopiujemy zawartość sync.css

  • Wchodzimy na stronę banku, logujemy się, na przeglądarce klikamy ikonkę CSS, wybieramy Open Styleboot...

  • Wklejamy zawartość sync.css i zapisujemy przyciskiem save.
